# EFF-pdf-processor
A pdf processor suite for error focus feedback in language teaching classrooms

1. The Dependencies List

    poppler-utils: Provides pdfseparate, pdftoppm, and pdfunite.

    ImageMagick: Provides montage and convert.

    pdftk-java: Provides pdftk (the tool for stamping letterheads).

    pdfarranger: For the initial PDF GUI review.

    GIMP: For finding crop coordinates.

    Xournal++: The final annotation tool.

    GNU Coreutils: (Standard on Mint) Provides shuf, mkdir, etc.
